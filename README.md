Bidget - E-Commerce Pot Bekas
    Deskripsi :Website Bidget merupakan platform e-commerce untuk penjualan pot bekas (second-hand plant pots) yang masih layak pakai. Dibangun menggunakan Next.js dan Tailwind CSS, aplikasi ini memiliki fitur lengkap untuk user dan admin dalam mengelola produk serta transaksi.

    Tentang Project : Project ini dibuat untuk mendukung gaya hidup ramah lingkungan dengan cara memanfaatkan kembali pot bekas yang masih bisa digunakan. Melalui website ini, pengguna dapat: Melihat produk pot bekas Menambahkan produk ke keranjang, Dan Melakukan checkout.


Fitur Utama
A. Untuk User
    - Melihat daftar produk pot bekas
    - Detail produk
    - Menambahkan ke keranjang
    - Checkout produk
    - Riwayat transaksi
B. Untuk Admin
    - Dashboard admin untuk monitoring website
    - Tambah produk
    - Edit produk
    - Hapus produk
